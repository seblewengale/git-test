Responsive Web Design

Why do we have to use Responsive web design?

In designing the website every one should concern about the user .How can I make the sites accessable and useble is good quetion. Because now adays mobile Internet usage is increasing steadily, it’s extremely important that website is mobile friendly. Usually this isn’t a major concern. You have a website designed for desktop users and another site specifically developed for mobile users. But, is it possible to have a site that is equally favorable for both desktop and mobile users? When you have a website with responsive design, you can make changes quickly and easily. You do not need to worry about making changes on two websites. This flexibility is a huge advantage when you just want to make a quick design tweak fix a typo on your site—you only have to do it once.
